Description

有形如：ax^3+bx^2+cx+d=0的一个一元三次方程。

Analysis

一元多次方程，函数图像为穿针引线图。利用根在-100~100之间且相差>=1，对这个区间的整数进行枚举分治。以从左到右为顺序，直到出现f(l)·f(i)<0，则必有一根存在于(l,i)，对此区间进行二分查找，之后处理(i,r)。

这道水题让我错懵逼了，改了一下午，结果是a,b,c,d定义成整型了..
