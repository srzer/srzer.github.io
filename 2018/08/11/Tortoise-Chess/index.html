乌龟棋

Aug 11, 2018

Analysis

毋庸置疑的，对于棋盘上的每一个点都至多有四种转移方式，即分别用四种卡片到达此点。但是用卡片是有限制的，为了记录这个限制，只能开维记录下卡牌数，一记就是4维，既然记录各种卡牌数，那么格数维是可以算出来的，直接舍去。最后的答案，就是dp[c[1]][c[2]][c[3]][c[4]]。

Code O(sum^4)
