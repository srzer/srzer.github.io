求后序遍历

  • buer
  • 4 Minutes
  • September 2, 2018

Description

输入一棵二叉树(结点 <= 100)的先序和中序遍历序列，输出其后序遍历序列。

Analysis

遍历方式的区别是什么？是根。先序的根在最左，中序的根夹在子树中间，所以通过先序得到根，放到中序里又可以求出左子树和右子树，这样就产生了子问题..因为要求后序遍历，所以先处理两棵子树，再输出根。只要计算一下分治时先序和中序的哪一段是互相对应的就行了。
