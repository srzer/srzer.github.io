最长公共子序列

Aug 08, 2018 889

Description

给定2个序列X、Y，求X和Y的最长公共子序列Z的长度。

Analysis

对于序列x的第i个字符，序列y的第j个字符，只有两种情况：

  1. 相等，i和j对应最优，则dp[i][j]=dp[i-1][j-1]+1。
  2. 不相等，不考虑i或是不考虑j，则dp[i][j]=max(dp[i][j-1],dp[i-1][j])。

对于动规可行性判断：

  • 最优子结构：子公共序列最长才能使之后转移出的公共子序列最长
  • 无后效性：前面字符对应的选择与之后转移无关。

Code O(len^2)
