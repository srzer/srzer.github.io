最小花费

Sep 09, 2018 1267

Description

在n个人中，某些人的银行账号之间可以互相转账。这些人之间转账的手续费各不相同。给定这些人之间转账时需要从转账金额里扣除百分之几的手续费，请问A最少需要多少钱使得转账后B收到100元。

Analysis

为使钱数最少，则存留的百分比最高，依旧是最短路径。把最短路径里的加法改成了乘法，因为依旧满足结合律，所以没什么区别。数据为2000，稀疏图，Dijkstra和SPFA都适用。这里用了SPFA。

Code O(k·E)
