k-interval scheduling

Greedy algorithm:
Firstly sort the intervals in the increasing order of finish time.
Then we observe every interval by order, if choosing it doesn’t make any points contained in more than $k$ intervals we have chose, we choose this interval.
Proof of correctness:
From [Algorithm design] P167 (4.6), we know that the intervals we want can be divided into $k$ groups while in each group there is no intersection. Thus this problem can be viewed as $k$ machine dealing with these intervals while the intervals for one machine cannot intersect.
We can also view our greedy algorithm as: Observe every interval by order, if it intersects with all machines, abandon it; otherwise assign it to the valid machine with largest finish time.
Let $S_i$ be the result we get in greedy algorithm after we observe $I_1,…,I_i$. Statement: $S_i$ can be extended to an optimal result by scheduling $I_{i+1},…,I_n$. We prove this by induction on $i$.
Note that $i=0$ case is trivial. Suppose $S_i$ can be extended to an optimal result $S’$. Then observe $I_{i+1}$:
case 1. In greedy algorithm, we abandon $I_{i+1}$.
Then $S’$ can’t either choose $I_{i+1}$ as $I_{i+1}$ intersects with all machines. Thus $S_{i+1}$ can be extended to $S’$.
case 2. In greedy algorithm, we assign $I_{i+1}$ to the machine $r$.
case 2.1. $S’$ abandon $I_{i+1}$.
Then If $S’$ didn’t assign $I’$ which intersects with $I_{i+1}$ to machine $r$, $S’$ is not optimal, contradict! Thus, $S’$ assigns $I’$ to machine $r$, $I’$ intersects with $I_{i+1}$. As the finish time of $I_{i+1}$ is smallest, $I’$ can only be one interval. Then we can replace $I’$ with $I_{i+1}$ in $S’$, it remains optimal. Thus $S_{i+1}$ can be extended to an optimal result.
case 2.2 $S’$ assign $I_{i+1}$ to $r’$.
If $r’=r$, then $S_{i+1}$ can be extended to $S’$. If $r’\ne r$, then we know that: in $S_i$, machine $r$ has larger finish time than machine $r’$, thus we can exchange the intervals$\in {I_{i+2},I_{i+3},…,I_n}$ that belongs to machine $r$ with the intervals$\in {I_{i+1},I_{i+2},…,I_n}$ that belongs to machine $r’$ in $S’$. It remains optimal. Thus $S_{i+1}$ can be extended to an optimal result.
Then we know that $S_n$ is an optimal result. Thus the greedy algorithm is correct. $\square$

二重叉积证明：
$$
c\times (a\times b)
$$
is clearly on the plane $ab$

Let
$$
c\times (a\times b) = ua+vb
$$
Then we have
$$
0 = u a·c+vb·c
$$
Thus
$$
u:v = b·c:a·c
$$
Then
$$
c\times (a\times b) = k(b·c a+a·c b)
$$
之后呢？还没想到。

但是可以用更简单的方法：
$$
\bar a\times (a\times b)= a (\bar a·b)\
\bar b\times(a\times b) = -b(\bar b·a)
$$
即证。
