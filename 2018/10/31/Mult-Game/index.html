乘法游戏

  • buer
  • 4 Minutes
  • October 31, 2018
Description

乘法游戏是在一行牌上进行的。每一张牌包括了一个正整数。在每一个移动中，玩家拿出一张牌，得分是用它的数字乘以它左边和右边的数，所以不允许拿第1张和最后1张牌。最后一次移动后，这里只剩下两张牌。你的目标是使得分的和最小。

Analysis

水题里还算有价值的一道。如果枚举第一次取的纸牌再分裂区间会有后效性，不如枚举最后一次选的纸牌避免后效。
