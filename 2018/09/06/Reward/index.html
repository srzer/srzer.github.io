Description

由于无敌的Wts在2005年世界英俊帅气男总决选中胜出，Hz Company总经理Mr. C心情好，决定给每位员工发奖金。公司决定以每个人本年在公司的贡献为标准来计算他们得到奖金的多少。

于是Mr.C下令召开m方会谈。每位参加会谈的代表提出了自己的意见：“我认为员工a的奖金应该比b高！”Mr.C决定要找出一种奖金方案，满足各位代表的意见，且同时使得总奖金数最少。每位员工奖金最小为100元。

Analysis

拓扑排序..确定拓扑顺序后将奖金逐层加1。
