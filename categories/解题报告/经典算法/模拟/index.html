    友好数对

    Description

    在顺利完成繁重的家庭作业后，小y 感到非常的厌倦和无聊。于是，他随手列出了N 个数，这些数中有些数对他是喜欢的，有些数对他是不喜欢的。

    他喜欢的数对叫做“友好数对”，如果两个数至少有一个相同的数字(不要求在相同的位置），那么这两个数就是友好数对。现在，请你帮助小y 在这N 个数中找出有多少友好数对。
