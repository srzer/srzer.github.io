数的划分

Sep 20, 2018 503
Description

loj10018

将整数n分成k份，且每份不能为空，问有多少种不同的分法。

Analysis

n个数分为k份，常规思路：dp[n][k][last]=Σdp[n-i][k-1][i]，状态和复杂度冗余。不妨每次选择第一个数的大小i，然后将所有位置都减去i，产生更简洁的子问题。

Code O(nk)
