合唱队形

Aug 11, 2018

Analysis

题目要求一个凸字型队列，左半部分为单调上升，右半部分为单调下降，为使此凸字形队列最长，则使以峰为终点的最长上升子序列加上以峰为起点的最长下降子序列的和最大，那么很明显，只要做两次单调子序列动规即可。

利用我们在LIS优化中得出的结论，可以轻松优化至O(nlogn)。

Code O(nlogn)

边界很重要。
