扩展二叉树

Sep 2, 2018

Analysis

因为中序和后序都要输出，所以最好进行建树。

扩展前光知道先序是建不出树的，因为知道根的位置却无法分出左右子树。扩展之后为什么就可以确定了呢？因为空节点被标记之后，叶子节点就可以被确定，也就确定了左子树在什么地方结束。如果左子树的根为空，那就直接确定，反之继续递归子问题，只要得到子问题的右子树结束点即可。所以还要再同理递归求出右子树的结束点并返回其值。
