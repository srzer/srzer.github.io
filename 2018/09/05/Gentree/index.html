家谱树

  • buer
  • 5 Minutes
  • September 5, 2018

Description

有个人的家族很大，辈分关系很混乱，请你帮整理一下这种关系。给出每个人的孩子的信息。输出一个序列，使得每个人的后辈都比那个人后列出。

Analysis

建立一个有向无环图，每次搞掉入度为0的点，更新他所能到达的，把新增的入度0的点塞队列里，这就是拓扑排序。
