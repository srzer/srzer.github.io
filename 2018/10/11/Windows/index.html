Description

给你一个长度为N的数组，一个长为K的滑动的窗体从最左移至最右端，你只能见到窗口的K个数，每次窗体向右移动一位。

你的任务是找出窗口在各位置时的max value,min value。

Analysis

单调队列模板题。对于两个数字，既靠右又较大的一定是最优的（对于maxv而言），而靠右却较小也是值得保留的，因为靠左的会提前过期。所以需要记录的只是一个单调递减队列，每次调控过期队头即可。stl::deque非常适合。
