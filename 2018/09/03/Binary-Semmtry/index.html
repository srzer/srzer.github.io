对称二叉树

  • buer
  • 3 Minutes
  • September 3, 2018

Description

如果二叉树的左右子树的结构是对称的，即两棵子树皆为空，或者皆不空，则称该二叉树是对称的。给出二叉树的按层次顺序遍历，编程判断是否对称.

Analysis

为了便于判断，先把二叉树扩展成完全二叉树，即把缺的补上‘#’(因为输入时末尾的空节点不会记入)。

因为是按层次遍历的完全二叉树，所以每个根的俩孩子都是相邻的，只需要循环每一层，判断每一组兄弟是否都为空和不空即可。
