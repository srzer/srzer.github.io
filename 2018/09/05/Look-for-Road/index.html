寻找道路

9月 05, 2018 2010

Description

在有向图G 中，每条边的长度均为1 ，现给定起点和终点，请你在图中找一条从起点到终点的路径，该路径满足以下条件：

  • 路径上的所有点的出边所指向的点都直接或间接与终点连通。
  • 在满足条件1 的情况下使路径最短。

请你输出符合条件的路径的长度。

Analysis

一开始可以先从终点宽搜一波传递闭包，复杂度O(n)。将所有到不了终点的点打上标记，再把所有与其相连的点打上另一种标记。这时候所有点都可以到达终点了，做一个最短路径就行。

因为图比较稠密，所以用dijkstra。

Code O(nlogn)
