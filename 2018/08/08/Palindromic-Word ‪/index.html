回文词

Aug 8, 2018

Analysis

对于回文很明显不能用线性动规写了。对于第i个和第j个字符(i<j)，存在两种情况。

  1. s[i]=s[j]，则i在回文中对应j最优，dp[i][j]=dp[i+1][j-1]
  2. s[i]<>s[j]，要么插入s[i]，要么插入s[j]，dp[i][j]=min(dp[i+1][j],dp[i][j-1])+1

动规可行性判断：

  • 最优子结构：处理i,j之前的子状态花费最小才能使(i,j)最优
  • 无后效性：先前插入字符后形成的一定是回文字符串，不会影响之后的选择。

Code O(n^2)
