最大权路径

  • buer
  • 11 Minutes
  • November 1, 2018
Description

给定一个n个点m条边有向图，每个点有一个权值，求一条路径，使路径经过的点权值之和最大。你只需要求出这个权值和。

允许多次经过一条边或者一个点，但是，重复经过的点，权值只计算一次。

Analysis

默认边权非负，显然需要缩点。转换成DAG拓扑排序Dp一下即可。
