点的距离

Description

LibreOJ 10130

给定一棵 n 个点的树，Q 个询问，每次询问点 x 到点 y 两点之间的距离。

Analysis

模板题。可以清晰地看出两者各自的优点，Tarjan快(599ms)，倍增简单(916ms)。

Tarjan O(n+q)
树上倍增 O(n+qlogn)
