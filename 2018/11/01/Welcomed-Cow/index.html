最受欢迎的奶牛

  • buer
  • 8 Minutes
  • November 1, 2018
Description

每头奶牛都梦想成为牛棚里的明星。被所有奶牛喜欢的奶牛就是一头明星奶牛。所有奶牛都是自恋狂，每头奶牛总是喜欢自己的。

奶牛之间的“喜欢”是可以传递的——如果A喜欢B，B喜欢C，那么A也喜欢C。牛栏里共有N 头奶牛，给定一些奶牛之间的爱慕关系，请你算出有多少头奶牛可以当明星。

Analysis

传递闭包问题，不难发现强连通的奶牛相当于一只奶牛，Tarjan缩点，如果只存在一个出度为0的SCC，那么该分量内所有奶牛皆为明星奶牛。
