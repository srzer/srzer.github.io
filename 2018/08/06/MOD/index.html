幂取模

Aug 06, 2018 606

Description

求a的b次方模k的值，a,b及k*k均是小于231的正整数。

Analysis

利用x^k^=(x ^i^)^(k/i)^，很容易想到通过修改a和k，快速求出a^b^，而将k不断的除以一个数，除以2无疑是高效的。每次把k/2，a就取二次幂，一旦2无法整除k，再利用x·x^k-1^=x^k^，将ans乘上一个此时的a，再做上述处理。
