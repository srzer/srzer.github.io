循环比赛

Aug 07, 2018 1102

Description

设有n个选手进行循环比赛，其中n=2m，要求每名选手要与其他n-1名选手都赛一次，每名选手每天比赛一次，循环赛共进行n-1天，要求每天没有选手轮空。

Analysis

要做到每行每列都有1~9，类似数独但是没有宫的限制。样例提示非常明显：

  • 第一行到第二行进行了以1为单位的交换。
  • 第二行到第三行进行了以1为单位的交换后再进行以2为单位的交换。
  • 第四行到第五行进行了上述交换后又进行了以4为单位的交换。

那么达到题目要求的矩阵需满足：每2^i^的倍数行的下一行就要针对上一行进行长度为2^i^单位之间的调换。直接暴力模拟：O(m·n^2^)
