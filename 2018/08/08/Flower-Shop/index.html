花店橱窗布置

Aug 8, 2018

Analysis

对于花瓶i，花束j而言有两种情况。

  1. 放，则dp[i][j]=dp[i-1][j-1]+s[i][j]
  2. 不放，则dp[i][j]=dp[i-1][j]

动规可行性判断：

  • 最优子结构：之前的花摆放美学值最大才能使之后的状态最优
  • 无后效性：做出选择后并不会妨碍之后花束的放置。

Code O(m·n)
